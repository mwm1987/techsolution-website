Demo de celda robotizada y control PLC

Gemelo digital de una estación automática de paletizado

Esta demostración representa la interacción entre un PLC, sensores, actuadores, cinta transportadora y un robot industrial de manipulación. El objetivo es visualizar en tiempo real el comportamiento de una secuencia de control y sus principales variables de producción.

¿Cómo funciona este gemelo digital?

1. Entrada y Detección

Las cajas ingresan por la cinta transportadora. El sensor óptico detecta la presencia del objeto en la zona de recogida y detiene el motor de la cinta, iniciando un temporizador de retardo (TON).

2. Lógica del PLC Ladder

El controlador PLC ejecuta constantemente su lógica Ladder. Cuando se verifica el temporizador y se cumplen los enclavamientos de seguridad, se activa la salida física que comanda la ejecución del robot.

3. Cinemática y Ventosa

El brazo robótico se despliega utilizando cálculos de cinemática inversa. Al llegar al objeto, activa la electroválvula de vacío (ventosa), asegura la caja y realiza el traslado para apilarla en el pallet.

Tecnologías aplicadas

PLC Ladder Engine

Lógica de control discreto industrial simulada en tiempo real por un motor lógico javascript.

Cinemática Inversa

Cálculos trigonométricos de articulaciones para posicionar de forma exacta el cabezal del robot.

Canvas 2D

Animaciones vectoriales fluidas a 60 FPS con trazado físico de piezas móviles y cajas.

Digital Twin

Representación virtual interactiva idéntica a una celda física de producción real.
